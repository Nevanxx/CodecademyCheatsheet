TABLE ATTRIBUTES

Tag Name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The set of rows defining the head of the table
<tbody> Table Body The set of rows defining the body of the table
<td> Table Data The table row container
<tfoot> Table Footer The set of rows defining the footer of the table
Attribute Name Description
cellpadding cellpadding Defines the space between the cell content and its border.
cellspacing cellspacing Sets the space between individual table cells.
border border Specifies the width of the table border.
height height Specifies the height of the table.
width width Specifies the width of the table.
